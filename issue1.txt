1. Дан код:

var a = 1, b = 1, c, d;
c = ++a; alert(c);           // 2
d = b++; alert(d);           // 1
c = (2+ ++a); alert(c);      // 5
d = (2+ b++); alert(d);      // 4
alert(a);                    // 3
alert(b);                    // 3

Решение
c = ++a; alert(c);           // 2
Переменнная a инициализирована значением 1, префиксный инкремент возвращает значение переменной после увеличения, поэтому переменной c  будет присвоено значение 1+1=2 и переменная а получит значение 2.
d = b++; alert(d);           // 1
Переменнная b инициализирована значением 1, постефиксный инкремент возвращает значение переменной до увеличения, поэтому переменной d будет присвоено значение b до уваеличения, т.е. 1, после этого будут выполнен инкремент, т.е. b=2.
c = (2+ ++a); alert(c);
В строке 4 переменная а в результате инкремента получила значеение 2, здесь выполняется еще один префиксный  инкремент, в результате а получает значение 3, к этому значекнию прибавляется 2, получаем 5.
d = (2+ b++); alert(d);    //4
В строке 5 переменная b  после инкремента получает значние 2. Постфиксный инкремент в строке 7 вернет значение 2, оно сложится с 2, результат (4) будет присвоен переменной d, затем знапчение переменной b  увеличится 1 , b=3/
alert(a);                    // 3
alert(b);                    // 3
После двух операций инкремента переменные a и b получают значения 3. В данном случае не важно префиксный или постфиксный инкремент  выполнялся.
